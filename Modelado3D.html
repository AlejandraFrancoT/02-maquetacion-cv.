Modelado 3D

Estos proyectos fueron elaborados a partir de las técnicas con primitivas y además con la aplicación de materiales, sus correspondientes nudos, texturas, incluyendo la parte de esculpido para lograr algunos detalles, cada uno de estos proyectos fueron realizados a nivel academico, mediante revisiones y mejoras

Modelado de crema

Para este proyecto, se elaboró principalmente en un entorno para modelado 3D, conocido como Blender. Se planteó enfocada en un producto comercial, sin embargo la idea inicial sería que esta tuviera un bote o técnicamente llamado “tarro” o “frasco cosmético”, por lo que eso nos dejaba varias opciones, podría ser una crema facial o algún producto con esa forma. En la cual su presentación fuera de esa manera. Así que se realizó un modelado lo más preciso y realista que se pudiera, además de poder ubicarlo en un entorno de forma profesional como si se tratará de un anuncio


Proyecto integrador

Para este proyecto, se hizo uso de las técnicas aprendidas con anterioridad, en este proyecto se puede apreciar un entorno más sencillo pero haciendo resaltar los objetos principales, pero logrando una armonía para que de esta forma ningun elemento quedara fuera o desequilibrara la escena, usando una paleta de colores en tonos beige y formales
